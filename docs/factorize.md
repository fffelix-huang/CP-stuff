## 質因數分解
```cpp
long long a;
vector<long long> factors = factorize(a); // 回傳 a 的質因數 (排序)
```

## 因數
```cpp
long long a;
vector<long long> divs = divisors(a); // 回傳 a 的所有因數 (排序)
```
